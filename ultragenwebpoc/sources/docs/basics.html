UltraGen uses a inline-token syntax to setup the parts where you want to insert some data from a source. The syntax is very simple: just put a "{{" to start a piece and a "}}"

     
    <-- Simple example -->
    <div class='{{ userField }}'>
        <p>Your name: {{ name }}</p>
        <p>Ninckname: {{ lower(username) }}<p>
        <p>Birthplace: {{ @remoteBirthPlace }}<p>
    </div>

Note:

You can use another symbol but must follow the pattern "two symbols". If you want to use "<" to open and ">" to close, in the template you must write "<<" and ">>".
